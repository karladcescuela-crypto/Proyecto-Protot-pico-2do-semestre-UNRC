# Proyecto-Prototipico-2do-semestre-UNRC
En este repositorio se encuentra material que se ocupo para la realizacion del proyecto prototipico "Suelos Inteligentes" de 2° semestre del equipo 1 de la Licenciatura en Ciencia de Datos para Negocios de la Universidad Nacional Rosario Castellanos [UNRC].
